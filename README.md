# 🎓 SEBA Akademi – YKS Hazırlık Platformu

Modern, kullanıcı dostu ve etkileşimli bir TYT–AYT hazırlık platformu.

## 📖 Proje Hakkında

SEBA Akademi, üniversite sınavına hazırlanan öğrenciler için geliştirilmiş kapsamlı bir eğitim platformudur. Platform; konu anlatımları, video dersler, konu bazlı testler, ilerleme takibi ve öğrenci paneli gibi özellikleri tek bir sistem altında sunmaktadır.

Amaç, öğrencilerin TYT ve AYT süreçlerini dijital ortamda daha verimli yönetmelerini sağlamak ve öğrenme deneyimini kişiselleştirmektir.

## ⚙️ Kurulum

Projeyi bilgisayarınıza klonlayın:

```bash

```

Klasöre girin:

```bash
cd yks-hazirlik-platformu
```

Ana sayfayı çalıştırın:


veya VS Code Live Server kullanın.
